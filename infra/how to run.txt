Run these in separate terminals from the repo root.

Apply Supabase migrations first:
- supabase/migrations/004_passive_pipeline.sql
- supabase/migrations/005_passive_evidence_meta.sql

1. Redis
infra\run-redis.bat

(Manual: docker start civx-redis — or docker run -d --name civx-redis -p 6379:6379 redis:7-alpine)

2. Backend
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

3. AI pipeline workers (Redis + 5 workers — preflight, logs, GPU warmup)
infra\run-ai-pipeline.bat

Check status (YOLO / LocateAnything / worker heartbeats):
infra\run-ai-status.bat
Or: http://127.0.0.1:8000/api/system/pipeline-status

Stop workers:
infra\run-ai-stop.bat

Legacy alias: infra\run-workers.bat (same as run-ai-pipeline.bat)

4. Build web for Caddy
cd web
npm run build

5. Caddy
infra\run-caddy.bat

6. Cloudflare tunnel
infra\run-tunnel.bat

--- All-in-one hosting (API + workers + Caddy + tunnel) ---
infra\run-hosting.bat

Public URL:
https://civx.xrnozy.me

Local dev only:
cd web
npm run dev

Orphan temp file cleanup (after crashes):
cd backend
python scripts/cleanup_orphan_frames.py --max-age-hours 24
